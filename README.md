# 🚀 Link na Bio.

Este projeto é uma página de **Link na Bio** desenvolvida para centralizar meus principais links, projetos e redes sociais de forma profissional, rápida e estilizada. O foco foi criar uma interface responsiva, com uma estética moderna focada em *Dark Mode*.

## 💡 Por que ter um Link na Bio próprio?
Sua marca é única, por que usar o mesmo layout de milhares de outras pessoas? 

O Linktree padrão tira o usuário do seu site e leva para a plataforma deles. Com um Link na Bio próprio, o cliente fica no seu domínio, o que melhora o seu ranqueamento no Google.

## 🎨 Design & Estética
Seguindo meu padrão de desenvolvimento, a interface utiliza:
* **Dark Mode:** Fundo escuro para maior conforto visual.
* **Neon Accents:** Detalhes em [insira a cor: Verde Neon/Roxo] para criar destaque e uma identidade visual tecnológica.
* **Responsividade:** Layout otimizado para dispositivos móveis e desktop.

## 🛠 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização com foco em design responsivo.
* **JavaScript:** Interatividade e animações.

## 📋 Funcionalidades
- [ ] Centralização de links profissionais sob domínio próprio.
- [ ] Botões com efeitos de *hover* e transição.
- [ ] Ícones intuitivos para redes sociais e portfólio.
